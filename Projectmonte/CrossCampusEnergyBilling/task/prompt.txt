Generate monthly energy billing for 25 facilities, 5 campuses (NORTH_CAMPUS/SOUTH_CAMPUS/EAST_CAMPUS/WEST_CAMPUS/CENTRAL_CAMPUS), 4 utility zones, Jan-Jun 2024. Use YYYY-MM month format. facility_monthly_bill must have 150 rows: one per facility-month. Read all CSV input files from the task directory.
All round() means Python built-in round() (banker's rounding, round half to even).
Nulls: kwh null+peak not null: kwh=peak*8. Both null: carry forward last originally non-null kwh(only rows where kwh was present in the raw data update the carry-forward tracker; filled/imputed kwh values do NOT update it; default 0), peak=0. peak null+kwh not null: peak=round(kwh*0.15,2). kvar null: facility same-month median; if none 0. solar null: 0.
TOU: join day_classifications by zone+date for fracs+season. peak_kwh=round(kwh*peak_frac,3), shoulder/offpeak same. Rates from rate_schedules by zone+season+period. daily_cost=round(sum of period_kwh*rate per period,2).
Monthly: total_kwh=sum kwh. valid_reading_days=count non-null original kwh. energy_cost=sum daily costs. month_peak_demand=max peak_demand_kw.
Solar: solar_credit=round(min(total_solar,total_kwh)*(energy_cost/total_kwh)*0.80,2); 0 if total_kwh=0.
Demand ratchet per facility chronologically. Mo1: ratchet=month_peak. Later: prior_max=max peaks up to 3 prior months, ratchet=max(month_peak,round(0.80*prior_max,1)). floor=round(contracted*0.70,1). billed=round(max(ratchet,floor),1). ratchet_applied=YES if billed>month_peak else NO. demand_charge=round(billed*demand_charge_per_kw,2) for zone+season of last day, period=PEAK.
PF: apparent=round(sqrt(kwh^2+kvar^2),2), pf=round(kwh/apparent,4) if apparent>0 else 1.0. pf<min_pf: pts=round((min_pf-pf)*100,1), pf_adjustment=round(pts*penalty_pct*demand_charge/100,2). pf>=0.95: pts=round((pf-0.95)*100,1), pf_adjustment=-round(pts*bonus_pct*demand_charge/100,2). Else 0.
Equipment: per equip installed before month end, months_since=(month_start_year-install_year)*12+(month_start_month-install_month), eff=max(0.50,rated_eff-degrad*months_since) round 3. avg_eff=round(mean,3). <0.85: surcharge=round((0.85-avg_eff)*energy_cost*0.25,2). Else 0.
Overhead: class 1=5% 2=8% 3=12% 4=15%. base=round((energy_cost+demand_charge)*rate,2). class>=3: admin=round(energy_cost*0.03,2). overhead=round(base+admin,2).
total_bill=round(energy_cost+demand_charge+pf_adjustment+equipment_surcharge+overhead-solar_credit,2).
Cost grade(priority): CRITICAL=pf<0.75 AND billed>contracted*1.20. OVER_DEMAND=billed>contracted*1.30. LOW_EFFICIENCY=surcharge>0 AND pf<min_pf. PF_PENALTY=pf_adjustment>0. EFFICIENT=solar_credit>0 AND pf>=0.95 AND billed<=contracted. STANDARD=else.
consecutive_penalty_months: for current month per facility chronologically: if cost_grade is CRITICAL, OVER_DEMAND, LOW_EFFICIENCY, or PF_PENALTY then consecutive_penalty_months equals the number of immediately prior consecutive months that had one of those same four grades (0 for first penalty month); if cost_grade is EFFICIENT or STANDARD then consecutive_penalty_months is 0.
Facility Efficiency Score (FES): rolling score per facility chronologically, initial prev_fes=50.0. pf_input=power_factor*100. If cost_grade is EFFICIENT or STANDARD: fes=round(prev_fes*0.85+pf_input*0.15,3). If PF_PENALTY or LOW_EFFICIENCY: fes=round(prev_fes*0.60+pf_input*0.40,3). If CRITICAL or OVER_DEMAND: fes=round(min(prev_fes*0.50,40.0),3). No data: fes=round(prev_fes*0.90,3). Clamp to [0,100].
Out1 facility_monthly_bill.csv(150 rows): facility_id,month,total_kwh,valid_reading_days,peak_kwh,shoulder_kwh,offpeak_kwh,energy_cost,month_peak_demand,ratchet_demand,billed_demand,demand_charge,power_factor,pf_adjustment,solar_credit,equipment_surcharge,overhead_cost,total_bill,cost_grade,consecutive_penalty_months,facility_efficiency_score.
Out2 curtailment_credit_ledger.csv: per facility per event in that campus. baseline_kwh=mean kwh of last 10 qualifying prior days(weekdays excl zone holidays for facility zone or ALL, excl same-campus curtailment dates; fewer use available; none=0). actual_kwh=event day kwh post null-fill. reduction_kwh=round(max(0,baseline-actual),2). reduction_pct=round(reduction/baseline,4) if baseline>0 else 0. target_met=YES if reduction_pct>=target else NO. credit_amount=round(reduction*credit_rate,2) if YES else 0. Cols: facility_id,event_date,baseline_kwh,actual_kwh,reduction_kwh,reduction_pct,target_met,credit_amount.
Out3 campus_quarterly_summary.csv: campus,quarter(integer 1 or 2),facility_count,active_facilities,total_kwh,total_energy_cost,total_demand_charges,total_solar_credits,total_curtailment_credits,total_bills,avg_power_factor,avg_fes=round(mean of facility_efficiency_score for last month of quarter across active facilities,3),cost_efficiency_rate=round(EFFICIENT|STANDARD in quarter last month/active,4). active_facilities=unique facilities with >=1 reading in quarter. avg_power_factor=mean pf of facility-months with kwh>0 round 4.
Out4 demand_ratchet_tracker.csv: facility_id,month,month_peak_demand,prior_3mo_max,ratchet_demand,contracted_floor,billed_demand,ratchet_applied. prior_3mo_max=0 for first month.
Out5 facility_cost_profile.csv: per facility. months_active=months with readings. Cols: facility_id,campus,utility_zone,demand_class,months_active,total_kwh,avg_monthly_bill,min_monthly_bill,max_monthly_bill,final_billed_demand,total_demand_charges,total_solar_credits,total_pf_adjustments,worst_cost_grade,final_fes,cost_category. final_billed_demand=last month billed_demand. worst_cost_grade=highest priority cost_grade. final_fes=facility_efficiency_score from last month. cost_category: HIGH_COST if final_billed>contracted*1.20 OR >=3mo billed>contracted*1.20. ELEVATED if final_billed>contracted OR >=2mo pf_adjustment>0. OPTIMIZED if total_pf_adjustments<0. NORMAL else.
