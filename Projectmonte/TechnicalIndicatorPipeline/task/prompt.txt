Compute technical indicators for the OHLCV price data in prices.csv and generate trading signals with portfolio performance metrics.
Calculate these indicators using the exact formulas specified. EMA uses the formula EMA_t = price_t * alpha + EMA_{t-1} * (1 - alpha) where alpha = 2 / (period + 1) and the first EMA value equals the SMA of the first period prices. SMA is the simple arithmetic mean of the last period prices. RSI uses Wilder smoothing where alpha = 1 / period and RS = avg_gain / avg_loss with the first average calculated as SMA of the first period changes and subsequent averages as prev_avg * (period - 1) / period + current_change / period. ATR uses True Range defined as max(high - low, abs(high - prev_close), abs(low - prev_close)) with the first ATR as SMA of the first period True Ranges and subsequent ATR using Wilder smoothing. MACD equals EMA_12 minus EMA_26 with Signal as EMA_9 of MACD and Histogram as MACD minus Signal. Bollinger Bands use SMA_20 as middle band with upper and lower bands at middle plus or minus 2 times the population standard deviation of the last 20 closes. Stochastic_K equals (close - lowest_low_14) / (highest_high_14 - lowest_low_14) * 100 and Stochastic_D equals SMA_3 of Stochastic_K. ADX requires computing Plus_DM as max(high - prev_high, 0) if high - prev_high > prev_low - low else 0 and Minus_DM as max(prev_low - low, 0) if prev_low - low > high - prev_high else 0 then Plus_DI as 100 * Wilder_smooth(Plus_DM, 14) / ATR_14 and Minus_DI as 100 * Wilder_smooth(Minus_DM, 14) / ATR_14 then DX as 100 * abs(Plus_DI - Minus_DI) / (Plus_DI + Minus_DI) and finally ADX as Wilder_smooth(DX, 14).
Generate BUY signals when MACD crosses above Signal while RSI is below 70 and close is above SMA_50. Generate SELL signals when MACD crosses below Signal while RSI is above 30 and close is below SMA_50. A crossover occurs when the current difference has opposite sign from previous difference.
Output indicators.csv with columns date, close, SMA_20, SMA_50, EMA_12, EMA_26, RSI_14, ATR_14, MACD, Signal, Histogram, BB_Upper, BB_Middle, BB_Lower, Stoch_K, Stoch_D, Plus_DI, Minus_DI, ADX, signal where signal is BUY or SELL or HOLD. Round all numeric values to exactly 4 decimal places. Rows before sufficient data for an indicator should have NaN for that indicator.
Output performance.csv with columns metric and value containing total_signals as count of BUY plus SELL signals, buy_signals as count of BUY, sell_signals as count of SELL, avg_rsi_at_buy as mean RSI when BUY signal occurs, avg_rsi_at_sell as mean RSI when SELL signal occurs, signals_above_bb_upper as count of signals when close > BB_Upper, signals_below_bb_lower as count of signals when close < BB_Lower, avg_adx_at_signal as mean ADX when any signal occurs, first_signal_date as the date of first BUY or SELL, last_signal_date as the date of last BUY or SELL, avg_atr_at_signal as mean ATR when any signal occurs, max_histogram as maximum Histogram value, min_histogram as minimum Histogram value, and days_between_signals as average trading days between consecutive signals rounded to 2 decimals.
