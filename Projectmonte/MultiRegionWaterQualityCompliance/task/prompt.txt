Generate quarterly water quality compliance for 30 stations, 5 regions (NORTH/SOUTH/EAST/WEST/CENTRAL), Jan-Jun 2024. station_compliance must have 180 rows: one per station-month incl zero-reading months. Use YYYY-MM format for month columns (e.g. 2024-01). Read all CSV input files from the task directory.
Nulls: ph/turbidity/chlorine null=carry forward last valid for station, default caution. lead null=0. coliform null=exclude from coliform PDI only. flow_rate null=offline, exclude from quality calcs. readings_count=all readings. valid_readings_count=flow_rate_lpm not null.
PDI per param per online reading: m<=caution:0. caution<m<=violation:(m-caution)/(violation-caution)*0.5. violation<m<=critical:0.5+(m-violation)/(critical-violation)*0.5. m>critical:1.0.
CQI=100*(1-weighted_mean_PDI). Weights ph=0.15 turbidity=0.20 chlorine=0.15 lead=0.30 coliform=0.20. Coliform avg excludes null coliform. Clamp 0-100, round 2 dec.
RCS per station chronologically. First data month:RCS=CQI. CQI>=80:prev*0.85+CQI*0.15. 60<=CQI<80:prev*0.6+CQI*0.4. CQI<60:RCS=CQI. No data:prev*0.95. Round 3 dec each step.
risk_trend: compare CQI to last data-month CQI. IMPROVING cqi>prev+2. DECLINING cqi<prev-2. STABLE otherwise. First data:NEW. No data:INSUFFICIENT.
Violations priority: CRITICAL_REPEAT=max PDI>0.9 AND FAIL inspection within 90 days before month-end inclusive. CRITICAL=max PDI>0.9. MULTI_VIOLATION=>=3 params avg PDI>0.5. HEALTH_HAZARD=lead+coliform avg PDI both>0.5. SINGLE_VIOLATION=any avg PDI>0.5. WATCH=any avg PDI>0. COMPLIANT. No readings:NO_DATA.
Tiers: CRITICAL_REPEAT=5 CRITICAL=4 MULTI_VIOLATION/HEALTH_HAZARD=3 SINGLE_VIOLATION=2 WATCH=1 rest=0. base_amount from penalty_rates where month-start in effective range. adj_factor from regional_adjustments by region+quarter(1.0 if null). workday_fraction=business_days/calendar_days round 4; business_days=weekdays minus holidays matching station region or ALL. consec_months for current month: if current tier<2 then consec_months=0; if current tier>=2 then consec_months=number of immediately prior consecutive months that had tier>=2 (0 for first violation month). penalty_amount=base_amount*adj_factor*workday_fraction*(1+0.15*consec_months) round 2. maint_credit=min(maint_cost*0.05,penalty_amount*0.20) round 2; maint_cost=sum(parts_cost+labor_cost) from maintenance_log for station+month. net_penalty=max(0,penalty_amount-maint_credit).
station_compliance.csv: station_id,month,readings_count,valid_readings_count,avg_pdi_ph,avg_pdi_turbidity,avg_pdi_chlorine,avg_pdi_lead,avg_pdi_coliform,cqi,rcs,violation_class,risk_trend,consecutive_violations,penalty_amount,maintenance_credit,net_penalty.
regional_summary.csv: region,quarter,station_count,active_stations(>=1 valid reading),total_readings,valid_readings,avg_cqi,avg_rcs,violations_count(tier>=2),critical_count(CRITICAL+CRITICAL_REPEAT),total_net_penalties,compliance_rate=(stations with data in quarter last month AND rcs>=70)/active_stations round 4.
anomaly_report.csv: station_id,reading_date,parameter,measured_value,threshold_type,threshold_value,pdi_value. parameter values are the source column names: ph, turbidity_ntu, chlorine_mg_l, lead_ppb, coliform_count. Per param per online reading where PDI>0.5. CRITICAL/VIOLATION/CAUTION by exceeded limit.
penalty_ledger.csv: per station-month with tier>0. station_id,month,violation_class,violation_tier,base_amount,adjustment_factor,workday_fraction,consecutive_months,gross_penalty,maintenance_cost,maintenance_credit,net_penalty.
station_risk_profile.csv: per station. station_id,region,zone_tier,months_monitored(with valid readings),total_readings,avg_cqi(r2),min_cqi,max_cqi,final_rcs(last month),total_violations(tier>=2),total_net_penalties(r2),worst_violation(highest-tier class),risk_category:HIGH_RISK if rcs<60 or violations>=4,ELEVATED if rcs<75 or violations>=2,MODERATE if rcs<90,LOW_RISK.
